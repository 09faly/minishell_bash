# minishell_bash

A custom shell implementation in C, mirroring basic Bash functionalities.

## Build and Run

```bash
make
./minishell
```

## How to run

1. Build the project:
   ```bash
   make
   ```
2. Launch the shell:
   ```bash
   ./minishell
   ```
3. Run commands as you would in bash. Use `exit` or `Ctrl+D` to quit.
